==============================================================
README: LINKING NET OVERSEAS MIGRATION WITH PROPERTY POSTCODE
==============================================================

Purpose:
This document explains about the R script and the results prepared for extracting Net Overseas Migration across NSW .
Data Collected:
NMO for 2018-19   : https://www.abs.gov.au/ausstats/abs@.nsf/mf/3218.0
NMO for 2017-18 : https://www.abs.gov.au/AUSSTATS/abs@.nsf/Lookup/3218.0Main+Features12017-18?OpenDocument

Comparison Data:
Data is against the collected Property Data.

Year: 
2019,2018

Process:
Using read_excel,'Table 1' from downloaded excel is extracted with specific fields of NOM,LGA,Year.

Output Format:
Year/LGA/NOM
 
Scripts and Output Files:
Prep_PostCode_ATO.R : Nom_LGA.csv
